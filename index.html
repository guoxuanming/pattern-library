  • Helper classes:
  • small-meta for 13px text
  • dim-el for light gray text (80% opacity)
  • fwb for bold text (500 weight)
  • fwl for light text (300 weight)
  • fsi for italics text
  • LESS variables:
  • @baseFontFamily for default font-stack – "Helvetica Neue", Arial, Helvetica, Verdana, sans-serif
  • @monoFontFamily for monospace font-stack – Consolas, "Lucida Console", Courier, monospace
  • @largeCopy for 18px text
  • @mediumCopy for 15px text
  • @smallCopy for 13px text
